❓ How to Play

Everything you need to get started and dominate the pitch.

Open this URL in your browser to play
http://localhost:3000
Make sure the server is running first: npm start

Step 1 — Start the Server

Run these two commands in your terminal once:

npm install
npm start

You'll see: ⚽ Soccer Manager running on http://localhost:3000

Step 2 — Open the Game

Navigate to the URL in any browser on the same machine:

http://localhost:3000

The game home screen loads. Fill in Username, Email, and an optional Team Name, then click Create / Load Manager.

Step 3 — Buy Coins

Go to Coin Store and buy a coin pack. Coins are used to open player packs.

Step 5 — Build Your Squad

Go to My Squad. Click + Add to Squad on your best players (max 11). Your squad's average rating determines your match results — always put your highest-rated players in!

Step 6 — Play Matches & Earn Coins

Go to Play Match → ▶ Kick Off! to play vs the AI.

Step 7 — Climb the Leaderboard

Check Leaderboard to see where you rank. Points: Win = 3 pts, Draw = 1 pt, Loss = 0 pts. Ties are broken by total goals scored.
